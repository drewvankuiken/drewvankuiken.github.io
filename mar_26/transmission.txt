# Slide(20):
Header: Grid Operator's Problem
Text:
$ \begin{multline*}
    \mathcal{L} = \sum_{r} \int_0^{q_{r\tau th}} c_\tau(x) dx + 
        \lambda_{th} (\sum_r D_{rth} - q_{rth}) + \\
        \sum_l \mu^l_{th} (k^l_t - \sum_r (q_{rth}-D_{rth}) SF^l_r)
\end{multline*} $

Columns:
- $c_{\tau}$ → Data
- $D_{rth}$ → Data
- $\mu^l_{th}$ → Data*
---
- $q_{r\tau th}$ → Choice Var.
- $\lambda_{th}$ → Model Output
---
- $k^l_{th}$ → Parameter
- $SF^l_r$ → Parameter*

# Slide(30):
Header: Entry Model - Entry to the Grid
Text:
- $N_{\tau}$ entrants appear, $N_{\tau}$ fixed to max number of counties of any tech entered in each $t$ in data $+ 1$
- Entry incurs $\nu_{r\tau t} + \varepsilon_{r\tau t}$
- Depends on fixed costs and expectations over state transitions
- **States:** $q_{r\tau t}$ - local capacity of each tech type, $Q_{\tau t}$ - grid-wide capacity of each tech type, $k^l_r$ - capacity of key constraints
- Implicitly, profits depend on how integrated region is. Higher $k$ means you compete against more firms, earn system lambda. Higher $q$ means congestion pricing lowers prices and profits

# Slide(31):
Header: equilibrium
Text:
- Entrants track own region state vars $q_{r\tau t}, k^l_{rt}$. Assume $k$ evolves according to own region quantity and aggregate quantities only
- Entrants assume aggregate quantities evolve according to AR-1 process
- Entry forms Moment-Based Markov Equilibrium (MME) (**Ifrach and Weintraub, 2017)

# Slide(32):
Header: Condensing the state space
Text:
- MME helps with curse of dimensionality
- Still ~80 counties with substantial generation, 300 transmission corridors
- Proposal: using most frequently congestion lines, condense generation into ~40 regions, connected by X transmission lines, let grid evolve for each region
- Problem parallelizable
