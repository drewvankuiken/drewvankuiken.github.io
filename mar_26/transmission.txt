# Slide(30):
Header: Entry Model - Entry to the Grid
Break:1
Text:
- $N_{\tau}$ entrants appear, $N_{\tau}$ fixed to max number of counties of any tech entered in each $t$ in data $+ 1$
- Entry incurs $\nu_{r\tau t} + \varepsilon_{r\tau t}$. Letting $\nu$ be distributed T1EV:
Break:1
Columns(top):
Choose 
$r \in \arg \max E_{\eta_r}[W(r,\tau,x_t)]$
Where
$W(r,\tau,x_t) = \rho E_t [\pi (x_{r\tau t})] - \phi_{r \tau}$
---
Yields logsum probability of $r$:
$
\begin{align}
p(r;\tau, x_t) = \frac{exp(W_r/\sigma^\eta)}{1 + \sum_{r^\prime \in R} exp(W_{r^\prime}/\sigma^\eta)}
\end{align}$

# Slide(31):
Header: Entry Model - Entry to the Grid
Text:
**States:** 
- $x_t$ = $q_{r\tau t}$ - local capacity of each tech type, $Q_{\tau t}$ - grid-wide capacity of each tech type, $k^l_r$ - capacity of key constraints
- Implicitly, profits depend on how integrated region is
- Higher $k$ → compete against more firms, earn system lambda
- Higher $q$ → congestion pricing lowers prices and profits

# Slide(31):
Header: equilibrium
Text:
- Entrants track own region state vars $q_{r\tau t}, k^l_{rt}$. Assume $k$ evolves according to own region quantity and aggregate quantities only
- Entrants assume aggregate quantities evolve according to AR-1 process
- Entry forms Moment-Based Markov Equilibrium (MME) (**Ifrach and Weintraub, 2017**)

# Slide(32):
Header: Consistency
Text: Dynamic competitive equilibrium:
- Expectations about evolution of $k$ and $Q$ are correct
- Entry probabilities given by logsum prob.
- Upgrades given by upgrade policy function
- Dispatch satisfies least-cost optimization conditional on grid topology

# Slide(32):
Header: Condensing the state space
Text:
- MME helps with curse of dimensionality
- Still ~80 counties with substantial generation, 300 transmission corridors
- Proposal: using most frequently congestion lines, condense generation into ~40 regions, connected by X transmission lines, let grid evolve for each region
- Problem parallelizable
